# hallo

Crystal project for [Make your own Shard in Crystal language](http://veelenga.com/make-your-own-shard-in-crystal-language) tutorial.

## Usage

```sh
$ git clone https://github.com/veelenga/hallo
$ cd hallo
$ make
$ ./bin/greeter
Hello, world ❗
$ ./bin/greeter "I :heart: Crystal"
I ❤️ Crystal
```
